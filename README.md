Цель этого проекта получить практические знания в разработке веб приложений при помощи фреймворка Django, изучить навыки работы с базой данных,
контейнеризации приложения. Реализовать базовый функционал приложения.
Серверная часть приложения написана на языке Python на фреймворке Django, пользовательская написана при помощи базового фронтенд стека (HTML/CSS).
Между собой они связывались посредством шаблонизатора Jinja, с помощью которого можно внедрять python-код напрямую в шаблоны.
Проект состоит из нексольких приложений. Реализовал регистрацию, аунтефикацию добавление товаров в корзину, а также добавление новых товаров через админ панель.
В качестве базы данных использовал PostgreSQL в связке с встроенной Django-ORM посредством написания моделей. Использовал Nginx в качестве маршрутизации и раздачи статических фйалов.
Законтейнизировал приложение в докере и поднял приложение при помощи docker-compose.
